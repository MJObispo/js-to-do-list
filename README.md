# To-Do List Page

A To-Do list page built with HTML, CSS/SCSS and Javascript 

### Features: 

<ol> 
<li><strong>CREATE</strong> tasks</li> 
<li><strong>READ</strong> or display them</li> 
<li><strong>UPDATE</strong> or edit them</li> 
<li><strong>DELETE</strong> them</li> 
</ol>
